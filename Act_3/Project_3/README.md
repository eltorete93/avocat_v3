# Libreria Nexus

![Nexus_logo](./public/Nexus.png)

## Objetivo del proyecto

El objetivo de este proyecto es desarrollar una aplicación web funcional construida con Vue que permita a los usuarios navegar y reservar espacios de coworking de la librería Nexus. La aplicación ofrece una navegación intuitiva, una interfaz de selección de espacios con filtros y una página de confirmación de reserva. El propósito es entregar una interfaz limpia, responsive y fácil de usar que simule un sistema real de reservas.

## Caracteristicas principales

- Aplicación desarrollada con Vue siguiendo buenas prácticas de desarrollo Front-End.

- Tres páginas principales:

  - Landing Page con información introductoria del servicio de coworking.

  - Página de selección de espacios donde se muestran todos los espacios disponibles y se pueden filtrar por planta o capacidad.

  - Página de checkout para revisar y confirmar la reserva.

- Menú de navegación presente en todas las páginas para moverse fácilmente por la aplicación.

- Diseño responsive, adaptado para escritorio, tablet y móvil.

- Contenido estructurado y realista, incluyendo textos, imágenes y distribución en filas y columnas.

- Maquetación centrada, con todo el contenido dentro de una columna central de 1200px para mejorar la visualización.

- Código organizado, incluyendo:
  - Archivo de enrutamiento separado.
  - Correcta nomenclatura de archivos y componentes.

- Aspecto profesional, simulando un sistema real de reservas listo para usar.

## Instalación y ejecución

Clonar el repositorio:

```bash
 git clone https://github.com/carloserick87/webcraft-frameworks.git
  ```

Instalar las dependencias:

```bash
npm install
```

Ejecuta el servidor de desarrollo:

```bash
npm run dev
```
